A minimalistic example of using the "async" keyword in Python 3.5+. First, you define a function with the "async" keyword (thus, "async def f()"). If you call such function via f(), you will get an object of the "coroutine" type. 
To run your coroutine, use the "run" function from the "asyncio" module.
